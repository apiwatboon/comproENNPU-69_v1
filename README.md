# สื่อการสอน: การเขียนโปรแกรมคอมพิวเตอร์ (ภาษา C)

เว็บสื่อการสอนแบบ Interactive รายวิชา **การเขียนโปรแกรมคอมพิวเตอร์** ภาษา C
สำหรับนักศึกษาสาขาวิศวกรรมคอมพิวเตอร์ โลจิสติกส์ และอุตสาหการ

## เนื้อหา
- บทเรียน 14 หน้า (สัปดาห์ที่ 1 รวม 3 คาบ → สัปดาห์ที่ 12)
- ตัวจำลอง/วิดเจ็ตประกอบการสอน และโค้ดที่กดรันได้จริงในเบราว์เซอร์
- ส่วน **📘 อ่านก่อนสอบ** สรุปประเด็นสำคัญก่อนทำแบบทดสอบทุกหน้า
- แบบทดสอบท้ายบทหน้าละ 10 ข้อ + คลังแบบฝึกหัด 110 ข้อ (`exercises.html`)

## เปิดดูในเครื่อง
```bash
python -m http.server 8741
```
แล้วเปิด <http://localhost:8741> (หรือดับเบิลคลิก `start-server.bat` บน Windows)

## เทคโนโลยี
HTML/CSS/JS ล้วน (ไม่มี build step) · GSAP, Chart.js ผ่าน CDN · คอมไพล์โค้ด C ออนไลน์ด้วย Wandbox API
