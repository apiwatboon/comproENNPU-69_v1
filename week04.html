🔧 ทำไมต้องภาษาซี?

ภาษาซีถูกสร้างโดย Dennis Ritchie ที่ Bell Labs ปี ค.ศ. 1972 เพื่อใช้เขียนระบบปฏิบัติการ UNIX — ผ่านมาห้าทศวรรษ มันยังคงเป็นรากฐานของแทบทุกอย่าง: ระบบปฏิบัติการ ไมโครคอนโทรลเลอร์ เครื่องจักรในโรงงาน และเป็น "ภาษาแม่" ที่ทำให้เรียนภาษาอื่นต่อได้ง่ายทั้งหมด

Program Structure

🧱 โครงสร้างโปรแกรมภาษาซี — บ้านที่ต้องมีครบทุกเสา

โปรแกรมซีทุกโปรแกรมในโลกมีโครงเดียวกัน: #include เรียกคลังคำสั่ง → int main() จุดเริ่มทำงาน → คำสั่งใน { } จบทุกคำสั่งด้วย ; — ลองรันโปรแกรมแรกในชีวิตเลย!

ตัวอย่างที่ 4.1 — hello.c
#include <stdio.h>      // เรียกคลังคำสั่งมาตรฐาน (มี printf อยู่ในนี้)

int main() {              // จุดเริ่มต้น — ทุกโปรแกรมเริ่มที่ main เสมอ
    printf("Hello, World!\n");
    printf("สวัสดีชาวโลก จากภาษาซี\n");
    return 0;             // บอกระบบว่าจบอย่างเรียบร้อย
}
⛔

เซมิโคลอน ; คือจุดจบประโยค

ทุกคำสั่งต้องปิดด้วย ; — ลืมตัวเดียว คอมไพเลอร์ฟ้อง error ทั้งโปรแกรมไม่ยอมรันเลย เป็นบั๊กอันดับหนึ่งของมือใหม่ภาษาซีทุกยุคทุกสมัย

🔤

ตัวเล็ก-ใหญ่ ไม่เหมือนกัน

ต้องพิมพ์ printf และ main ตัวเล็กเท่านั้น — พิมพ์ Printf หรือ Main คอมไพเลอร์ไม่รู้จักทันที

💬

คอมเมนต์ // และ /* */

ข้อความหลัง // คือโน้ตสำหรับมนุษย์ คอมไพเลอร์ข้ามให้ (หลายบรรทัดใช้ /* ... */) — เขียนอธิบายโค้ดให้เป็นนิสัยตั้งแต่วันแรก

Variables & Data Types

📦 ตัวแปรและชนิดข้อมูล — กล่องที่ต้อง "เลือกขนาด" ก่อนใช้

ตัวแปรคือกล่องเก็บข้อมูลในหน่วยความจำ (RAM จากบทที่ 2!) — จุดสำคัญของภาษาซี: ต้องประกาศชนิดก่อนใช้เสมอ ตามรูปแบบ ชนิดข้อมูล ชื่อตัวแปร = ค่า; เพื่อให้คอมไพเลอร์จองหน่วยความจำได้ถูกขนาด

ชนิดเก็บอะไรตัวอย่างประกาศตัวกำหนดรูปแบบ
intจำนวนเต็ม (นิยมมากที่สุด)int age = 20;%d
floatจำนวนจริงมีทศนิยมfloat height = 175.5;%f
doubleทศนิยมความละเอียดสูงdouble pi = 3.14159265359;%lf
charตัวอักษร 1 ตัว (ใน ' ')char grade = 'A';%c

⭐ สังเกต: char ใช้ เครื่องหมายคำพูดเดี่ยว 'A' เก็บได้ทีละตัวอักษรเท่านั้น ส่วนข้อความยาว ๆ ใน " " จะได้พบในบทสตริง — และภาษาซี ไม่มีชนิด bool ในตัว ใช้ int แทน (0 = เท็จ, อื่น ๆ = จริง)

ตัวอย่างที่ 4.2 — about_me.c
#include <stdio.h>

int main() {
    int age = 19;          // จำนวนเต็ม
    float gpa = 3.45;      // ทศนิยม
    char grade = 'B';      // ตัวอักษรเดี่ยวใน ' '
    age = age + 1;         // วันเกิด! ค่าใหม่ทับค่าเดิม
    printf("อายุ %d ปี เกรด %.2f ระดับ %c\n", age, gpa, grade);
    return 0;
}

สังเกตบรรทัด age = age + 1; — เครื่องหมาย = คือ "กำหนดค่าให้" ไม่ใช่เท่ากับทางคณิตศาสตร์: คำนวณฝั่งขวาก่อน (19 + 1 = 20) แล้วเก็บกลับเข้ากล่อง

ตัวอย่างที่ 4.3 — circle.c (double กับงานคำนวณละเอียด)
double r = 2.5;
double area = 3.14159 * r * r;
printf("รัศมี %.1f เมตร\n", r);
printf("พื้นที่วงกลม = %.2f ตร.ม.\n", area);

📏 กฎการตั้งชื่อตัวแปรภาษาซี

✅ ใช้ตัวอักษรอังกฤษ a–z, A–Z, ตัวเลข และ _ ได้
✅ ขึ้นต้นด้วยตัวอักษรหรือ _ เท่านั้น
❌ ขึ้นต้นด้วยตัวเลขไม่ได้ (2score)
❌ ห้ามเว้นวรรค ห้ามใช้ - และห้ามตัวอักษรไทย
❌ ห้ามซ้ำคำสงวน เช่น int, return, if, for
💡 ธรรมเนียมนิยม: ตัวเล็กคั่นด้วย _ เช่น total_price

Output with printf

🖨️ printf และตัวกำหนดรูปแบบข้อมูล — ปากเสียงของโปรแกรม

จะแสดงค่าตัวแปร ต้องบอก printf ว่าข้อมูลชนิดอะไร ผ่าน ตัวกำหนดรูปแบบข้อมูล (format specifier): %d จำนวนเต็ม · %f ทศนิยม · %c ตัวอักษร · %s ข้อความ — และ \n ขึ้นบรรทัดใหม่

ตัวอย่างที่ 4.4 — format.c
int stock = 42;
float weight = 7.25;
char zone = 'B';
printf("สินค้าคงเหลือ %d ชิ้น\n", stock);
printf("น้ำหนักรวม %f กก.\n", weight);
printf("น้ำหนักรวม %.2f กก.\n", weight);
printf("เก็บที่โซน %c\n", zone);
printf("%s\n", "พร้อมจัดส่ง");

🔬 เครื่องเลือกตัวกำหนดรูปแบบ

พิมพ์ค่าตามแบบภาษาซี (เช่น 42, 3.5, 'A', "hello") แล้วดูว่าต้องประกาศชนิดอะไรและใช้ % ตัวไหน:

— พิมพ์ค่าเพื่อตรวจ —
⚠️

จับคู่ให้ถูก ไม่งั้นได้ค่ามั่ว

ใช้ %d กับตัวแปร float (หรือสลับกัน) โปรแกรม "ไม่พัง" แต่แสดง ค่ามั่ว ออกมาเงียบ ๆ — อันตรายกว่า error เพราะรายงานผิดโดยไม่มีใครเตือน จำง่าย ๆ: int→%d · float→%f · char→%c · ข้อความ→%s และคุมทศนิยมด้วย %.2f (2 ตำแหน่ง)

ตัวอย่างที่ 4.5 — escape.c (อักขระพิเศษ \n และ \t)
printf("รายการ\tจำนวน\tราคา\n");
printf("น็อต M6\t200\t150.00\n");
printf("แหวนรอง\t500\t275.50\n");
printf("--------------------------\n");
printf("\\n คือขึ้นบรรทัดใหม่ \\t คือเว้นแท็บ\n");
From Flowchart to Code

🧮 ตามสัญญา: ผังงาน BMI → โปรแกรมภาษาซี

สัปดาห์ที่แล้วเราวาดผังงาน BMI ไว้ วันนี้มันกลายเป็นภาษาซีแล้ว! ใส่น้ำหนัก–ส่วนสูง แล้วกด "รันทีละบรรทัด ▶" ดูโปรแกรมไล่ทำงานเหมือนเดินผังงานทีละกล่อง (คำสั่ง scanf รับค่าและ if เลือกทาง จะเจาะลึกในบทที่ 5–6 — วันนี้ดูภาพรวมก่อน)

bmi.c
1float weight, height, bmi;
2scanf("%f", &weight);
3scanf("%f", &height);
4bmi = weight / (height * height);
5printf("BMI = %.2f\n", bmi);
6if (bmi >= 25)
7 printf("น้ำหนักเกิน\n");
8else printf("น้ำหนักปกติ\n");

📝 แบบทดสอบท้ายบท (10 ข้อ)

สามจุดที่ข้อสอบชอบ: โครงสร้างโปรแกรม · จับคู่ชนิดข้อมูลกับ % · เซมิโคลอน

📘 อ่านก่อนสอบ · สรุปประเด็นสำคัญ

สรุปก่อนทำแบบทดสอบ: ทบทวนโครงสร้างโปรแกรม ชนิดข้อมูล และตัวกำหนดรูปแบบ แล้วลองทำ 10 ข้อ

1

โครงสร้างโปรแกรมภาษาซี

ทุกโปรแกรมเริ่มทำงานที่ฟังก์ชัน main() และต้องมี #include <stdio.h> เพื่อใช้ printf/scanf

#include <stdio.h>
int main() {
    printf("Hello");
    return 0;
}

2

ชนิดข้อมูลพื้นฐาน

int = จำนวนเต็ม · float = ทศนิยม · char = อักขระ 1 ตัว (ใช้ '...' ครอบ)

int age = 25;
float gpa = 3.45;
char grade = 'A';

3

ตัวกำหนดรูปแบบ (format specifier)

จับคู่ชนิดข้อมูลกับสัญลักษณ์ใน printf: %d=int, %f=float, %c=char · %.2f = ปัดทศนิยม 2 ตำแหน่ง

float price = 49.987;
printf("%.2f", price);   // 49.99

4

เซมิโคลอน ; และกฎตั้งชื่อตัวแปร

ทุกคำสั่งต้องจบด้วย ; มิฉะนั้น คอมไพล์ไม่ผ่าน · ชื่อตัวแปร ห้ามขึ้นต้นด้วยตัวเลข และห้ามมีเว้นวรรค (เช่น 2name ผิด, name2 ถูก)

💡

จุดที่ข้อสอบชอบถาม

3 จุดยอดฮิต: ลืม ; → error · จับคู่ % ผิดชนิด · ชื่อตัวแปรขึ้นต้นด้วยเลข — ระวังให้ดี
